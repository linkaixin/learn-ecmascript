DOM：Document Object Model 文档对象模型
宿主对象：由浏览器提供一系列方法
JavaScript下有三种对象
1. 本地对象：Native Object
Object、Function、Array、String、Number、Boolean、 Error、Date、RegExp
2. 内置对象：Built-in Object
Global、Math
Global是一类对象的统称，例如：isNaN() parserInt decodeURI encodeURI;
Global的属性有：Infinity NaN undefined
本地对象和内置对象都是ECMA的内部对象，是由ECMA提供，与浏览器没有关系
3. 宿主对象：Host Object
执行JS脚本的环境提供的对象，也就是浏览器对象，由于浏览器之间的实现方式不一样，所以会出现兼容性问题
浏览器对象包含Window(BOM)和document(DOM)
DOM其实是在Window下面的，但是DOM有W3C规范，而BOM没有
DOM：通过浏览器提供的这一套方法表示或者操作HTML和XML
XML -> XHTML -> HTML
