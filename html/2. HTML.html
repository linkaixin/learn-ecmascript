HTML:超文本标记语言 HyperText Markup Language

HTML 不是一种编程语言，而是一种标记语言（markup language）。

根据标签的不同分为单标签与双标签

双标签

xhtml规范标准：

  • 1. html标签必须使用小写
  • 2. 属性名必须使用小写
  • 3. 属性至使用双引号包裹
  • 4. 单双标签都需要闭合

是 HTML 文档中的一个声明，用于告诉浏览器这个文档使用的是哪个 HTML 版本。这个声明应该位于 HTML 文档的第一行。

根标签 lang属性的值代表不同的语言和方言 常用语言：en(英文),zh-CN表示中国大陆的简体中文,zh-Hans(简体中文)，zh-Hant - 繁体中文

电商网站

title:

主页：网站名称 + 主要的关键字/关键词的描述 详情页：详情名称 + 网站名称 + 简介 列表页：分类名称 + 关键字 + 网站名称 文章页：标题 + 分类 + 网站名称

keywords:

100个字符，使用,分割，网站名称 + 分类信息

description：描述信息 80-120汉字，综合title + keywords的简单描述

搜索引擎认知的优先级：title > description > keywords
